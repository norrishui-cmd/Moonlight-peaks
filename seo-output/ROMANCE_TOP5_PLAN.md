# Romance Top 5 Implementation Plan

## 1. `/romance`
- **Cluster:** Roster / who can you romance (largest romance sub-cluster, ~1,526 impressions)
- **Clicks/Impr/CTR/Position:** 317 / 4,133 / 7.67% / 7.10
- **Problem:** Already the clear primary route and performing reasonably, but under-links to the
  two best-performing romance-adjacent pages on the site (children FAQ at 10.06% CTR, gift finder
  at 15.62% CTR) — visitors landing here for the roster don't get pointed to the strong follow-up
  content.
- **Proposed changes:** add contextual internal links from within the existing step-by-step
  content to the marriage FAQ, children FAQ, and gift finder tool at the points where those topics
  are already discussed (no new sections, no title/H1 change).
- **Internal links:** → `/faq/can-you-get-married-in-moonlight-peaks`, → `/faq/can-you-have-children-in-moonlight-peaks`, → `/tools/romance-gift-finder`.
- **Expected impact:** Medium (largest volume page in the cluster; link equity + potential
  assisted-conversion to the FAQs, without touching what's already working).
- **Confidence:** High.

## 2. `/previews/romance-and-marriage-what-we-know`
- **Cluster:** Marriage / general romance
- **Clicks/Impr/CTR/Position:** 0 / 2 / 0% / 9.5 (thin volume, but a clear factual bug)
- **Problem:** dek and body text say "before the July 7 launch" / "ahead of launch" — the game has
  been out for 2 days. Stale pre-launch framing on a page whose entire premise is "what's
  confirmed."
- **Proposed changes:** fix the stale phrases to reflect post-launch status; no structural change.
- **Internal links:** link to `/romance` as the fuller, currently-maintained source.
- **Expected impact:** Low-Medium (low current volume, but a real accuracy fix, and low effort).
- **Confidence:** High.

## 3. `/faq/can-you-get-married-in-moonlight-peaks`
- **Cluster:** Marriage
- **Clicks/Impr/CTR/Position:** not independently visible at page level this round (query-level:
  ~141 impressions across marriage-related queries).
- **Problem:** answer is accurate but thin ("Yes, you can marry a romance partner. There are no
  children in the game.") relative to how much more `/romance` itself already confirms about
  marriage mechanics (Heart Level 8, wedding within a week, spouse moving in) — using
  already-established, already-published facts, not new claims.
- **Proposed changes:** expand the answer with the same confirmed facts already live on `/romance`.
- **Internal links:** → `/romance`.
- **Expected impact:** Medium (matching the pattern of the children FAQ, which performs very well).
- **Confidence:** High.

## 4. `/tools/romance-planner`
- **Cluster:** Roster / planning
- **Clicks/Impr/CTR/Position:** 0 / 12 / 0% / 7.42
- **Problem:** Title ("Shortlist & Prep Tracker") still carries mild pre-launch "prep" framing;
  low but real 0%-CTR signal.
- **Proposed changes:** lighten the title to reflect present-tense in-game use rather than launch
  prep (small wording change, not a full rebuild — this tool's intro copy was already updated in
  an earlier round; the title lagged behind).
- **Internal links:** ensure `/romance` links to it (verify/add if missing).
- **Expected impact:** Low (small volume) but zero-risk.
- **Confidence:** Medium.

## 5. `/tools/romance-gift-finder`
- **Cluster:** Gifts (romance-adjacent, distinct intent per Phase 8's own example)
- **Clicks/Impr/CTR/Position:** 10 / 64 / 15.62% / 6.78 — already strong (Bucket D: winner to defend)
- **Problem:** No problem with the page itself; the opportunity is purely that `/romance` doesn't
  link to it despite covering gift-giving conceptually.
- **Proposed changes:** none to this page directly — protect it. Action lives on `/romance`
  (item 1 above) and confirming this tool's own related-links are intact.
- **Internal links:** verified existing links out are intact; added the inbound link from
  `/romance`.
- **Expected impact:** Low direct change, but protects and reinforces an already-good performer.
- **Confidence:** High.
